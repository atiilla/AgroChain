This panel is designed to demonstrate blockchain resilience against cyberattacks. By tampering with a block, you can observe how the SHA-256 hash chain is broken.

How SHA-256 Hash Works

Block(n).hash = SHA256( index + timestamp + data + Block(n-1).hash + validator )

If 1 byte changes in any block, that block's hash becomes completely different.
This change invalidates the previous_hash field of all following blocks.

PoA validators detect this inconsistency immediately → CHAIN INVALID
